This example is for players of Extreme Startup who can't
get an HTTP server to run in their prefered development
environment. It shows how to do HTTP listening, regexp
in Java with tomcat.

To run :
mvn tomcat7:run

Update ExtremeStartup.java

The changes are taken into account after a few seconds (no need to restart)

To call it from a browser : http://localhost:8080/extreme-startup-tomcat?q=aze
